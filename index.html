Visualisation de Données pour le Diagnostic de l'Autisme

Contexte du Projet

Ce projet s'inscrit dans le cadre de la SAE 303, visant à utiliser l'eye-tracking pour identifier des bio-indicateurs discriminants dans le diagnostic de l'autisme. L'objectif est de développer un site interactif permettant la visualisation des données collectées.

Visualisation des Données

Les données recueillies sont affichées sous forme de graphiques interactifs permettant une analyse détaillée des zones explorées par les enfants lors du test d'eye-tracking.

Explorer les Visualisations

Résultats et Analyse

Les résultats montrent des différences significatives dans les comportements visuels des enfants autistes par rapport aux enfants avec développement typique. Les zones telles que les yeux et la bouche sont particulièrement analysées.
